# EcoTrace | Aurora Sustainability Dashboard

## Project Overview
This project is a fully responsive frontend interface developed for the Week 2 and Single Page Application (SPA) simulation tasks. "EcoTrace" is designed as an enterprise-grade sustainability tracker, featuring a modern, dark-themed UI with glassmorphism effects and dynamic content rendering.

## Key Features
* Fully Responsive: Adapts seamlessly to mobile, tablet, and desktop viewports using a strict mobile-first CSS architecture.
* Dynamic Interactivity: Utilizes Vanilla JavaScript to simulate SPA behavior, revealing hidden dashboard modules without triggering page reloads.
* Modern UI/UX: Features the "Aurora" design system, utilizing CSS `backdrop-filter`, radial gradients, and fluid grid layouts.

## Tech Stack
* HTML5: Semantic structure for accessibility and SEO.
* CSS3: Custom styling, CSS Grid/Flexbox, and Media Queries (`@media`).
* Vanilla JavaScript: DOM manipulation and event listening.

## How to Run
Simply download the repository, extract the files, and open `index.html` in any modern web browser. No external dependencies or build tools are required.
